# 德宁之殿多层斗拱与横向侧楼梯设计

## 目标

继续依据实拍侧视关系完善德宁之殿：补齐首层左右侧斗拱，删除二层全部围护墙并用三层四面斗拱框架填充，保持牌匾为独立悬挂构件，同时将左右侧楼梯旋转 90° 后贴主台基前墙横向展开。

屋顶、鸱吻、大月台尺寸、主体台基高度与正面中央楼梯保持不变。

## 首层侧面斗拱

- 现有首层正面和背面柱头斗拱保留。
- 左右侧沿六进外围柱网增加完整柱头铺作。
- 左右侧每两个柱位之间增加补间铺作。
- 每侧共 `13` 组：`7` 组柱头铺作与 `6` 组补间铺作。
- 侧面铺作整体旋转 90°，出跳方向朝向檐口。
- 左右侧斗拱最高点低于下层屋面，最低点与首层侧梁相接。

## 二层无墙斗拱框架

- 删除 `upperEnclosure` 中正面、背面、左侧、右侧的全部红色围护板和竖棂。
- 二层继续保持零立柱。
- 在原二层围护高度内设置下、中、上三层斗拱框架。
- 每层均沿正、背、左、右四面连续布置完整铺作和补间铺作。
- 下层斗拱尺寸略小，中层为标准尺寸，上层出跳最大；三层之间使用连续梁枋连接。
- 每层正背面各 `15` 组、左右侧各 `9` 组，共 `48` 组；三层合计 `144` 组。
- 三层使用 `userData.tier = 'lower' | 'middle' | 'upper'` 标记。
- 最上层最高点低于上层屋面，最下层最低点高于下层屋面，不穿瓦、不悬空。

## 独立牌匾

- 保留现有无字木框、绿色底板、金色内边和垂花。
- 牌匾不作为任何围护墙或斗拱的子节点。
- 牌匾向正面外移，与正面斗拱带保持清晰间距。
- 增加一根顶部悬挂横梁和左右两根短吊杆。
- 横梁与上层正面梁枋相接，吊杆连接牌匾顶部，牌匾背面不接触斗拱。
- 牌匾继续保持无文字。

## 横向左右楼梯

- 正面中央楼梯保持当前方向和位置。
- 左右侧楼梯整体旋转 90°，运行轴由 Z 轴改为 X 轴。
- 两组楼梯与主台基正面墙平行，并贴近主台基前缘。
- 左楼梯高端连接大月台左侧，低端向建筑左侧下降。
- 右楼梯高端连接大月台右侧，低端向建筑右侧下降。
- 两组楼梯左右镜像，各保持 `10` 级。
- 台阶最低级与庭院地面相接，最高级与月台顶面相接。
- 侧楼梯两条坡栏同步转向 X 轴，主台基前栏在楼梯与月台连接处留口。

## 模块边界

- `src/model/timber-frame.ts`：首层侧斗拱、二层三层斗拱框架、删除二层围护、独立牌匾吊架。
- `src/model/timber-frame.test.ts`：数量、方向、层级、无墙和牌匾独立性测试。
- `src/model/foundations.ts`：横向侧楼梯和对应坡栏。
- `src/model/foundations.test.ts`：楼梯轴向、高低端、镜像与落地测试。

## 自动化验收

- 首层斗拱左右侧分别为 `13` 组，并同时存在柱头铺作与补间铺作。
- 二层围护板和围护竖棂数量均为 `0`。
- 二层斗拱共 `144` 组，每个 tier 包含四个方向。
- 二层继续没有立柱。
- 牌匾为 `grid` 的独立子节点，并包含横梁与两根吊杆。
- 牌匾包围盒与正面斗拱包围盒在 Z 轴上保持正间距。
- 左右侧楼梯包围盒的 X 向长度大于 Z 向宽度。
- 左楼梯最高级靠近月台左侧、最低级更靠左；右楼梯镜像相反。
- 三组楼梯仍各有两条坡栏。

## 浏览器验收

- 正面：二层红墙完全消失，可看见三层斗拱深度；牌匾独立悬挂在斗拱前方。
- 左侧：首层侧斗拱连续，二层三层侧斗拱无空缺。
- 鸟瞰：左右楼梯沿主台基前墙横向展开，高端接月台、低端向外。
- 旋转观察：牌匾不嵌入斗拱，楼梯和坡栏不穿入台基，斗拱不穿瓦。

## 非目标

- 不改变屋顶、正脊、鸱吻和瓦片。
- 不恢复二层墙面、二层立柱或牌匾文字。
- 不改变大月台约 `26m × 10m` 的尺寸。
- 不增加背面楼梯。
