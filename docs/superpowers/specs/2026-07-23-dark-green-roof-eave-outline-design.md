# 暗绿坡面与绿色檐口描边设计

## 目标

调整上下两层屋顶的分色关系：坡面基底使用暗绿色，筒瓦使用暗黄色，并在两层屋顶最外缘增加与屋脊一致的琉璃绿色描边。

## 材质

- 坡面基底：新增独立 `roofSurface` 材质，颜色 `#1A382C`，哑光、深于屋脊绿。
- 筒瓦：沿用 `tile` 材质 `#82794A`，保持褪色暗黄色。
- 瓦垄线：沿用 `tileRib` 材质 `#5A5737`。
- 檐口描边：直接复用 `glazedGreen`，即屋脊颜色 `#255942`。

## 几何

每层屋顶沿前、后、左、右四条最外檐口曲线生成细圆管描边。曲线沿现有翼角起翘轮廓采样，贴合屋面边缘，不新增悬空直线，也不改变屋顶尺寸和瓦片密度。

## 验证

- 单元测试锁定坡面、筒瓦、屋脊绿三种颜色。
- 单元测试确认上下两层各有四条檐口描边，且材质与屋脊共用。
- 运行全量测试和生产构建。
- 用桌面及手机视口检查颜色层次、描边贴合、画布非空和控制台错误。
