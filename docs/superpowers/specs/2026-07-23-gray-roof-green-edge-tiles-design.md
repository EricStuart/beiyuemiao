# 灰色屋面与绿色边沿瓦设计

## 目标

调整德宁之殿两层屋顶配色，使坡面和大部分基础瓦片呈现褪色灰色，同时保留并强化绿色琉璃构件的层次。此次只修改材质颜色和现有瓦片实例的颜色分区，不改变瓦片密度、朝向、屋顶几何或光照系统。

## 配色

- 坡面基层使用深灰色 `#4F534F`。
- 普通筒瓦使用褪色暖灰色 `#74736C`。
- 瓦沟线使用更深的灰色 `#565852`。
- 正脊、斜脊、鸱吻继续使用现有屋脊绿 `#255942`。
- 二层正面中央菱形区域继续使用绿色瓦片 `#2F543D`。

## 绿色边沿

绿色边沿直接通过现有 `InstancedMesh` 的实例颜色实现，不增加新对象：

- 两层屋顶四周檐口最外排瓦片保持屋脊绿。
- 每个坡面靠近斜脊的左右最外列瓦片改为屋脊绿。
- 绿色菱形区域继续使用独立的现有瓦片实例集合，不创建底板，避免穿模。

同一瓦片同时满足檐口和斜脊邻边条件时只应用一次绿色实例颜色。

## 验证

- 材质测试确认坡面、普通瓦和瓦沟线为指定灰色，菱形瓦及屋脊保持指定绿色。
- 实例测试确认檐口最外排和斜脊邻边最外列使用屋脊绿，并且未新增边沿网格对象。
- 浏览器检查桌面正面、侧面和手机视口，确认灰瓦分区清晰、绿色边沿连续、菱形区域可见且无穿模或控制台错误。
