# 鸱吻正面造型调整设计

## 参考目标

以用户提供的德宁殿鸱吻正面照片为造型依据，将现有抽象挤出轮廓改为多层真实几何。需要在建筑整体视距下清晰辨认出高背、卷尾、龙首和张口四个特征，旋转到斜视时也不能显示为平面贴图。

## 造型结构

- **高背主体**：使用有厚度的绿色挤出轮廓，下宽上窄，外侧背部近乎竖直，上缘略向屋脊内侧收拢。
- **卷尾**：从主体上部向屋脊内侧延伸，使用管状曲线形成 S 形上扬轮廓，末端以圆形卷首收束。
- **背鳍层次**：在主体上缘和外背设置至少6件圆角鳍片，轮廓逐步变小，形成照片中连续高起的鱼鳍节奏。
- **龙首**：头部位于主体下部并朝向正脊中心，绿色额头与金色吻部分层生成。
- **张口与装饰**：上颌、下颌分离，保留可见的口部空隙；增加两件角、浅色眼球、深色瞳孔和简化须饰。
- **底座**：继续使用厚重底座压住正脊端部，不改变现有屋顶标高、正脊长度和鸱吻安装点。

## 材质与对称

- 主体、卷尾和背鳍使用 `glazedGreen`。
- 吻部、上下颌、角和须饰使用 `gold`。
- 眼球使用 `stone`，瞳孔使用 `darkTimber`。
- 东西两件鸱吻仍由同一几何工厂生成，仅在 X 轴镜像；两件的包围盒尺寸必须一致。

## 验收标准

- 每件鸱吻包含一件主体、一件卷尾、至少6件背鳍、一件龙首、独立上下颌、两件角和眼部组件。
- 上颌和下颌的垂直间距大于 0.18m，正面可见张口轮廓。
- 卷尾顶部高于龙首，整体高宽比大于 1.35，保持参考照片的高耸比例。
- 鸱吻底座与正脊相交，整件鸱吻不超出正脊 X 向包围。
- 浏览器正面、斜前和侧面视角无穿模、无悬空，顶部轮廓左右镜像。
