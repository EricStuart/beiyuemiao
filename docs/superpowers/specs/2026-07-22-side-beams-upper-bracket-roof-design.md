# 侧立面梁枋、二层斗拱与上檐标高调整设计

## 背景

右视测绘图显示，首层柱头位置由连续的横向梁枋承托斗拱；二层檐下仍以密集、同模数的斗拱承接上檐，屋面不应与斗拱之间留下明显空带。当前模型的一层侧面缺少与正面相同的柱头梁枋，二层斗拱使用了额外的纵向放大比例，上层屋顶底标高偏高。

## 参考依据

- 用户提供的《曲阳北岳庙德宁殿横剖面图》右视测绘图。
- 测绘图用于约束柱头梁枋的连续关系、上下檐与斗拱的贴合关系，以及二层斗拱相对首层的模数比例；本轮不从低清图中强行读取新的绝对尺寸。

## 选定方案 A

1. **首层侧面梁枋**：在左、右侧面柱头位置补齐与正面相同的深色承重梁和绿色外枋，四面形成连续的柱头梁枋带；保留现有侧面斗拱和围护墙体。
2. **二层斗拱同模数**：移除 upper bracket 的 1.72/1.85 纵向拉伸，令二层和一层使用同一组四层斗拱几何尺寸与角色缩放。二层周长较小，因此采用相同的柱点/间柱点节奏，数量少于一层但间距密度保持一致。
3. **上檐下移贴合**：将上层屋顶底标高、二层承檐梁架、牌匾吊架作为同一标高组下移，使屋面檐底落在斗拱顶面附近；上层主脊高度同步下移，保持屋面坡度与鸱吻相对位置。
4. **牌匾约束**：牌匾继续独立悬挂并保持 9° 外倾；随承檐梁架一起下移，顶部不穿入上层屋面，底部仍位于两层屋面之间。

## 结构接口

- `addBracketSet` 的四层斗拱尺寸成为 lower/upper 共用的唯一模数。
- `createTimberFrame` 为左右侧面补充与正面等价的两道横板，并用共享标高常量计算上层承檐梁、吊架和牌匾位置。
- `createRoofs` 的上层 `baseY` 与 `ridgeY` 同步下移；下檐几何不变。

## 验证标准

- 左右侧面均存在深色梁和绿色外枋，且四道梁枋框架闭合。
- lower/upper 斗拱每组均为四层，upper 与 lower 的 stage 尺寸和缩放相同。
- 上层斗拱最高点与上层承檐梁架底面误差不超过 0.12m；上檐底面不悬空。
- 牌匾仍保持 9° 外倾，且包围盒顶部低于上层屋面底标高。
- 新增回归测试先失败后通过；全量测试、类型检查、生产构建和浏览器正/侧/俯视验收均通过。
