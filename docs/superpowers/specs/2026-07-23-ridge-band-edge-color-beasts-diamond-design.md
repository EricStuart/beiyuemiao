# 正脊饰带、坡面边带、走兽与菱形瓦设计

## 范围

只调整两层屋顶的装饰与颜色分区，不改变屋顶尺寸、瓦片密度、瓦片朝向、自然光循环或观察交互。

## 最高正脊黄琉璃饰带

在最高长方体正脊的正面增加一条独立薄饰带。饰带使用褪色黄琉璃色 `#B08D3E`，长度为正脊有效长度减去两端鸱吻避让区，高度约 `0.22` 米，厚度约 `0.08` 米。饰带略微凸出正脊正面，垂直居中，不穿入正脊或鸱吻。

新增 `yellowGlaze` 物理材质供饰带使用，保留现有 `gold` 材质用于小型金色细节。

## 坡面连续绿色边带

坡面基层直接使用顶点颜色分区，不添加覆盖条或重复网格：

- 每个坡面靠近两条斜脊的外侧边缘改为 `#2F543D`。
- 四周檐口最外缘的坡面基层同步改为 `#2F543D`。
- 坡面内部继续使用灰色 `#4F534F`。
- 现有檐口瓦和斜脊邻边瓦的实例颜色从屋脊绿改为菱形瓦绿 `#2F543D`，使瓦片和坡面边带成为同一颜色层。

顶点颜色直接乘入现有灰色坡面材质，边缘颜色使用相对于灰色基材的颜色倍率，确保最终显示颜色接近 `#2F543D`。

## 斜脊末端走兽

两层屋顶的四条斜脊在檐口末端各增加一只简化走兽，共 `8` 只。走兽复用现有中段脊兽的构件语言，但使用独立的 `ridge-end-beast` 类型、较小比例和末端定位。每只走兽底座贴住对应斜脊末端，并朝向檐口外侧。最高正脊两端继续只保留鸱吻，不叠加走兽。

## 横长菱形绿瓦

二层正面中央菱形调整为横长竖窄：

- 高质量模式下中心行约 `13` 片瓦，纵向共 `7` 排。
- 使用曼哈顿距离菱形掩码，横向半宽约 `0.3032`，纵向半高为 `3/14`。
- 菱形中心位于第七排瓦片中心附近。
- 左、右、上、下四个尖端各只保留 `1` 片瓦。
- 继续用瓦片实例替换原位置的灰瓦，不添加底板，避免穿模。

## 验证

- 材质测试确认黄琉璃材质为 `#B08D3E`。
- 模型测试确认最高正脊只有一条正面饰带，位置不超过正脊和鸱吻边界。
- 坡面测试确认使用顶点颜色，边缘色为 `#2F543D`，且没有新增绿色覆盖条对象。
- 屋脊测试确认上下两层各有 `4` 只末端走兽，共 `8` 只，并贴近对应斜脊末端。
- 菱形测试确认高质量模式下为 `43` 片瓦、横向 `13` 片、纵向 `7` 排，四个尖端各 `1` 片。
- 浏览器检查桌面正面、侧面和手机视口，确认饰带、绿色边带、走兽和菱形均可见，无穿模、闪烁、空白或控制台错误。
