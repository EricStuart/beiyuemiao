# 褪色黄琉璃瓦与菱形改色设计

## 目标

- 普通屋面瓦改为褪色、偏黄的旧琉璃瓦颜色。
- 二层正面中央菱形继续为绿色，但不再使用独立叠加面片。
- 上下两层屋顶的左、右坡面瓦片列数减半，正面和背面密度保持不变。
- 保持现有瓦片方向、坡面位置、屋脊颜色和屋顶几何不变。

## 实现

- 普通筒瓦使用低饱和黄绿色 `#82794A`。
- 瓦棱与瓦缝使用更暗的旧黄色 `#5A5737`，保留瓦面层次。
- 删除 `createGreenDiamondPatch` 生成的独立菱形几何。
- 继续用 `isGreenDiamond` 划分现有瓦片实例；菱形范围内的实例仅替换为现有绿色材质。
- 普通瓦与菱形瓦沿用同一套位置、坡面法线和表面偏移，避免穿模与闪烁。
- `collectTilePlacements` 按坡面选择列数：左、右坡面采用正面列数的一半，并继续包含两端边瓦，保证侧面均匀覆盖。

## 验证

- 材质测试锁定普通瓦、瓦棱与菱形瓦颜色关系。
- 屋顶测试确认不存在 `green-diamond-tile-bed` 独立面片，且绿色菱形瓦实例仍存在。
- 屋顶测试确认每个侧面瓦片实例数约为对应正面的一半，且上下两层规则一致。
- 完整测试、类型检查与生产构建通过。
- 在桌面和手机视口检查屋面颜色、菱形轮廓、瓦片贴合及控制台错误。
