# 屋顶坡面去线与暗灰黑瓦设计

## 目标

去除德宁之殿上下两层屋顶坡面上的辅助线条，并将普通灰白瓦调整为更暗的灰黑色，使屋面更沉稳，同时保留真实瓦片几何和现有绿色琉璃构件。

## 屋面线条

- 删除 `createTileLines` 生成的 `LineSegments` 辅助线对象。
- 上下两层屋顶均不再添加该线条对象。
- 保留 `roof-tile-covering` 中的半圆筒瓦实例，瓦片排列、碗口方向和表面法线方向不变。
- 保留屋面绿色檐口、斜脊边沿、中央菱形绿瓦、正脊和走兽。

## 瓦片颜色

- 普通瓦基础色从 `0x74736c` 调整为暗灰黑色 `0x44443f`。
- 屋顶坡面基层继续使用现有 `0x4f534f`，不随普通瓦一起压暗。
- `diamondTile`、`glazedGreen` 和 `yellowGlaze` 保持现有颜色。
- 绿色边沿瓦和菱形瓦继续通过实例颜色校正呈现目标绿色，不因普通瓦基础色变化而偏色。

## 测试与验收

- 材质测试确认普通瓦颜色为 `0x44443f`，其亮度低于原值和屋顶坡面基层。
- 屋顶测试确认上下两层均不存在坡面辅助线对象。
- 现有测试继续确认两层瓦片覆盖、绿色边沿和 `1、3、7、13、7、3、1` 菱形排列不变。
- 浏览器检查桌面和手机视图，确认没有残留辅助线、真实瓦片仍完整、绿色构件没有变色或消失。
