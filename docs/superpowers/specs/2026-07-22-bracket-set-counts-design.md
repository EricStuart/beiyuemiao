# 斗拱组数与柱网对应设计

## 目标

按实物观察修正德宁殿上下两层斗拱组数，同时保持每组斗拱与柱网、柱间的建筑逻辑。

## 布点规则

- 一层正面和背面各 26 组：10 个柱上组，最外两个柱间各1组，其余七个柱间各2组，共16个柱间组。
- 二层正面和背面各 22 组：柱上位置直接对应一层去除最外两根柱后的8个内柱位，七个柱间各2组，共14个柱间组。
- 一层左右侧面各 17 组：7 个柱上组，最外两个柱间各1组，中间四个柱间各2组，共10个柱间组。
- 二层左右侧面各 13 组：5 个柱上组，四个柱间各2组，共8个柱间组。
- 柱间一组时放在该间中点；柱间两组时放在该间的三分之一和三分之二位置，避免重叠并保持均匀。

## 结构约束

- 二层正面 X 轴柱位使用一层 X 轴柱网的内侧八点，二层侧面 Z 轴柱位使用一层 Z 轴柱网的内侧五点。
- 不改变现有斗拱四层几何、材质、缩放、屋顶和牌匾标高。
- 正背、左右必须镜像对称。

## 验收标准

- 自动测试分别验证四个立面的总数、柱上组数和柱间组数。
- 自动测试验证二层柱上斗拱坐标与一层内侧柱位一致。
- 浏览器正面和右侧视角下斗拱无重叠、无明显过大空隙，并且不与牌匾、屋顶穿模。
