# 台基、楼梯栏杆、菱形瓦与月台香炉鼎设计

## 目标

修正德宁之殿模型中台基底层外扩、正面台阶栏杆悬空、侧梯穿模及扶手重复断开的问题；同时保证各画质等级的绿色菱形瓦严格按指定行数排列，并在月台中央增加白色石质道教香炉鼎。

## 台基

- 删除主台基底部逐级外扩的下层和中层。
- 主台基改为一个从庭院地面延伸到石质压顶下方的垂直砖墙。
- 墙体平面尺寸固定为测绘尺寸 `48.03 x 31.77 m`，不再随高度外扩。
- 墙体使用原最底层地基层的深灰砖色；石质压顶和灰砖铺地保持现状。
- 前出月台继续保持 `25.10 x 19.86 m` 的凸字形平面及现有顶面标高。

## 正面台阶栏杆

- 两组坡栏向楼梯中心线内收，使栏杆横板、立柱底面全部位于 `6.4 m` 宽的踏步范围内。
- 坡栏仍沿踏步坡度布置，并保留上下端柱。
- 栏杆不得悬空于踏步外侧，也不得穿入月台墙体。

## 侧梯与扶手

- 左右侧梯维持现有横向走向、`4.8 x 3.0 m` 尺寸和十二级踏步。
- 每个侧梯仅保留外侧一组坡栏，删除靠主台基前缘的重复短栏杆和重叠立柱。
- 坡栏横板放在侧梯外缘踏步上方，不再偏离楼梯范围。
- 每组坡栏明确包含靠主台基方向端柱、靠月台方向端柱及两端之间的一条连续坡向横板。
- 靠月台端与月台侧栏杆使用同一个连接坐标；端柱可共点，但横板不得留缝或重叠。

## 绿色菱形瓦

- 不再使用依赖画质网格密度的连续比例掩码。
- 瓦片放置数据记录行号、列号、总行数和总列数，菱形按离散索引选择。
- 高、中、低画质都严格使用七行：`1、3、7、13、7、3、1`。
- 菱形继续位于上移后的屋面位置，四个尖端均为单片绿色瓦。

## 月台香炉鼎

- 香炉鼎位于月台中心轴线及月台平面中心附近，底部落在月台灰砖铺地上。
- 总高约 `2.3 m`，圆腹最大直径约 `2.2 m`，与大殿和月台尺度协调且不进入正面台阶范围。
- 造型由圆形口沿、收颈、大肚炉身、三足、双耳和顶部简化云纹饰件组成。
- 全部主体使用现有白色石材质，内部使用略深的石色表现炉口深度，不使用文字或彩绘。
- 香炉鼎作为独立场景组，便于测试位置、尺寸和后续调整。

## 测试与验收

- 自动测试确认主台基只有一个同尺寸砖墙主体，底层不再外扩。
- 自动测试确认正面坡栏包围盒完全落在正面楼梯横向范围内。
- 自动测试确认侧梯不与主台基墙体相交，每侧只有一组坡栏，端柱齐全，坡栏与月台栏杆连接点一致。
- 自动测试在高、中、低画质下确认菱形行数均为 `1、3、7、13、7、3、1`。
- 自动测试确认香炉鼎位于月台中心、落地且不与正面楼梯相交。
- 浏览器分别检查正面、左右侧面、俯视和手机视口，确认无悬空、穿模、重叠、断栏或控制台错误。
