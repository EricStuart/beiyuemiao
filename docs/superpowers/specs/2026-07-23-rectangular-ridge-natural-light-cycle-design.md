# 长方体正脊与自然光影循环设计

## 正脊

只调整上层屋顶最顶部的 `main-ridge` 主体。将横向圆柱体换成长方体，长度沿用现有正脊长度，竖直高度设为 1.2 米，进深厚度设为 0.52 米。长方体底部继续贴合屋脊，下方 `ridge-band` 装饰带、鸱吻和脊饰保留。

## 自然光

保留现有半球光和低强度环境光作为天空及地面反射。主方向光模拟太阳，在 90 秒内沿建筑周围的闭合椭圆轨道循环，轨道始终高于地平线，避免场景进入不可观察的黑夜。太阳高度变化会同步调整方向光强度、半球光强度以及暖冷色温，使阴影方向和长度连续变化。

中性测绘光照模式保持固定，不执行动态色温和强度更新；切回晴天模式后继续自然光循环。

## 验证

- 模型测试确认 `main-ridge` 使用 `BoxGeometry`，不存在圆柱形主脊，且竖直高度不少于 1.2 米。
- 纯函数测试确认日光周期闭合、位置始终在地平线上方、四分之一周期的位置和光强不同。
- 浏览器测试确认阴影与方向光诊断位置随时间变化，桌面和手机画布非空且无控制台错误。
